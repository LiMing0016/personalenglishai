阶段 3：第一次部署（未开始 ⬜）

目标：让后端服务第一次在公网稳定运行，并具备基础的可恢复能力。

任务清单

云服务器准备完成（VPS / 云主机，系统可 SSH 访问）

域名 api.personalenglishai.com DNS 解析完成（指向服务器公网 IP）

后端服务完成首次部署并稳定运行

HTTPS 配置完成（可通过公网安全访问）

服务支持异常退出或服务器重启后的自动恢复

完成标准（Exit Criteria）

通过公网地址 https://api.personalenglishai.com/api/ping 访问成功（HTTP 200）

服务器重启后，服务可自动恢复并再次访问成功

说明与约束（阶段冻结）

本阶段仅部署后端服务，不包含前端部署

不引入真实邮箱验证、第三方消息服务

不要求高可用、多实例或负载均衡

不做性能优化，仅保证稳定可访问

所有配置以“可复现部署”为优先目标

阶段交付物（Deliverables）

一个可公网访问的后端服务域名

一个可用于连通性验证的基础接口（/api/ping）

一套可重复执行的部署流程（支持再次部署/重启）

阶段完成判定

当上述 任务清单全部完成，且 完成标准全部满足，则判定：

阶段 3：第一次部署 —— 已完成 ✅
